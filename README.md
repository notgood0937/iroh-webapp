# Iroh P2P Password Manager WebApp

## Overview
This is a Password Manager WebApp project utilizing Iroh P2P technology and built on the Next.js framework. It allows users to securely store and manage their passwords in a decentralized manner.

## Features
- **Iroh P2P Integration**: Uses Iroh P2P for secure data storage and retrieval.
- **Next.js Framework**: Leveraging Next.js allows for server-side rendering and improved performance.
- **User Authentication**: Secure user authentication mechanisms to protect user data.
- **Responsive Design**: The web app is designed to be fully responsive, providing a seamless experience across devices.

## Getting Started
To get started with the project:
1. Clone the repository: `git clone https://github.com/notgood0937/iroh-webapp.git`
2. Navigate to the project directory: `cd iroh-webapp`
3. Install the required dependencies: `npm install`
4. Run the development server: `npm run dev`
